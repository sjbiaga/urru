# format for fold game (folding blocks unfold)

// line comment: grid size M (rows) by N (columns) (the board)
BOARD M N

-- yet another line comment: a shape - group of points (blocks)
BLOCK X1 Y2 X2 Y2 X3 Y3 ...
# ...

// empty cells at (X1, Y1), ...
EMPTY X1 Y1 ...
# ...

// wildcard cells
MULTI X1 Y1 ...
